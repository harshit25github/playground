# switch_prep

This folder is your full‑stack learning workspace (HTML/DOM → Node backend → React → Next.js).

## Where to start

- `playground/` → structured, step‑by‑step mini projects
- `js/` and `html/` → your existing notes / experiments

## Quick run

- HTML/DOM: open `playground/00-html-dom/index.html` in your browser
- JS lessons: start at `playground/js/README.md`
- DOM lessons: start at `playground/dom/README.md`
- CSS lessons: start at `playground/css/README.md`
- Node backend (no deps): `cd playground/10-node-http-api` then `node server.js`
- Windows shortcut: `playground/scripts/node-api.cmd`
