# JavaScript track

Each folder is a focused concept + exercises. Run the `.js` files with Node:

```sh
node exercise.js
```

## Lessons

- `00-basic-playground/` - systematic examples + activities (basic)
- `01-basics/` → types, coercion, conditions, loops
- `02-functions-scope/` → functions, scope, closures
- `03-arrays-objects/` → arrays, objects, common patterns
- `04-this-and-prototypes/` → `this`, binding, prototype basics
- `05-async/` → promises, async/await, error handling
- `06-sync-vs-async/` → event loop, microtasks, timers, patterns (detailed)
- `07-advanced-playground/` - advanced patterns + async control flow
- `08-oop-playground/` - OOP exercises
