# DOM track

Each folder is a browser exercise. Open `index.html` and complete the TODOs in `app.js`.

## Lessons

- `01-selectors-events/` → select elements + handle events
- `02-rendering/` → render lists from state
- `03-event-delegation/` → handle clicks for dynamic lists
- `04-forms-validation/` → form handling + validation UI
- `05-localstorage/` → persist UI state
- `06-fetch-api/` → call your Node API from the browser
- `07-async-in-dom/` → timers, promises, async/await, debounce, abort/timeout + fetch
- `08-insta-post-skeleton/` → Instagram-like post UI (skeleton only)
- `09-events-playground/` → most-used DOM events (examples)
- `10-virtual-keyboard-skeleton/` → on-screen keyboard UI (skeleton)
- `11-keyboard-piano-skeleton/` → piano UI + key mapping (skeleton)
- `12-parent-child-nodes/` → parentNode/childNodes vs children (playground)
- `13-reels-bubbling-skeleton/` → event bubbling/delegation practice (skeleton)
