# DOM 07 - Async in DOM (timers, promises, fetch)

Is folder ka goal: browser (DOM) me async concepts practice karna:

- `setTimeout` / `setInterval`
- Promises (`then`) vs timers order (microtask vs macrotask)
- `async/await` flow
- **sequential vs parallel** async work
- **debounce** (typing wale scenario)
- **timeout** + **abort** (cancel a pending async task)
- Optional: backend `fetch()` (`playground/10-node-http-api/`)

## Run

1. (Optional) backend start:
   - `cd playground/10-node-http-api`
   - `node server.js`
2. Open `index.html` in your browser.
