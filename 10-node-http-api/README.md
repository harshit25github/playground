# 10 - Node HTTP API (no dependencies)

## Run

```sh
node server.js
```

Then open:

- `http://localhost:3000/` (welcome page)
- `http://localhost:3000/health`
- `http://localhost:3000/api/todos`

## What you practice

- Node `http` server
- routing by method + path
- JSON request/response
- basic CORS
